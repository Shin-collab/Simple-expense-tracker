โปรแกรมนี้ใช้สำหรับบันทึกและแสดงผลค่าใช้จ่าย โดยสามารถแยกตามหมวดหมู่ และดูย้อนหลังตามช่วงเวลาได้  
มีฟังก์ชันการกรองข้อมูล (Filter) และแสดงผลแบบ Dashboard

# ไฟล์ที่ใช้
- 01_index.html : หน้าแรกของโปรแกรม
- 02_dashboard.html : หน้าสรุปค่าใช้จ่าย
- 03_style.css : การตกแต่งหน้าตาเว็บไซต์
- 04_script.js :  การกรองข้อมูล

# วิธีใช้งาน
เปิดไฟล์ `01_index.html` ด้วยเว็บเบราว์เซอร์เพื่อเริ่มใช้งาน
